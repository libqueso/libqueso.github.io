QUESO

A C++ library for the Quantification of Uncertainty for Estimation, Simulation and Optimization

The QUESO library

QUESO stands for Quantification of Uncertainty for Estimation, Simulation and Optimization.

QUESO is a collection of algorithms for solving statistical inverse problems with quantified uncertainties, and is designed for flexibility, portability, usability and extensibility. Its software design follows an object-oriented approach and its code is written in C++ over MPI, enabling the use of multiprocessor environments.

Installation

You can obtain QUESO tarballs here.

If you do not have a configure script in the top level directory, run ./bootstrap to generate a configure script using autotools. After this is done, QUESO may be installed in the 'usual' fashion as shown here:

$ ./configure
$ make -j8  # Pass -j8 to use 8 processors to build queso
$ make install

Additional options may be provided to the configure script if desired; run ./configure --help for details.

Dependencies

At a minimum, QUESO compilation requires MPI and linkage against two external libraries:

  • Boost
  • GSL

QUESO also has several optional dependencies that enable additional functionality:

  • Teuchos
  • GRVY
  • HDF5
  • GLPK
  • PETSc

Should you be interested in using the optional infinite dimensional capabilities of QUESO, then you also need the following dependencies:

  • libMesh compiled with SLEPc
  • HDF5

Contributing

Contributions are very welcome. If you wish to contribute, please take a few moments to review the branching model QUESO utilizes.

Support

If you have questions or need help with using or contributing to QUESO, feel free to ask questions on one of the mailing lists:

  • queso-users mailing list for questions regarding usage and reporting bugs
  • queso-dev mailing list for discussion regarding development of QUESO
